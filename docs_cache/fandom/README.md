# Roblox BedWars Fandom Cache

This folder is for local cache files generated from the Roblox BedWars Wiki on Fandom.

Run:

```powershell
python maintenance/refresh_fandom_cache.py
```

Use `--include-text` only when you need full plain-text extracts locally. Generated JSON cache files are ignored by git because Fandom content is community-authored CC-BY-SA material and should keep source attribution.
